Xbox 一直在下载游戏服务，无法安装游戏？

发表于 2023-07-02 更新于 2024-08-29 分类于 Games

昨天下午买了官方 (Microsoft) 的 Minecraft，今天在 Windows 10 上安装显示需安装游戏服务，我按下”下载”后下方弹出”正在安装游戏服务。这将需要几分钟的时间。“也就是上面红框的部分。

于是我就等待，结果过了五分钟，还没安装好。于是我打开”设置“ → “应用“，发现游戏服务只有16KB。卸载按钮是灰色的，无法卸载。

于是我去 Xbox 官网寻找答案，结果真被我找到了。

方法如下

  1. 选择任务栏上的”搜索”图标，键入”PowerShell”，右键单击”Windows PowerShell”，然后选择”以管理员身份运行”。或者 Windows 徽标键 + X，选择”Windows PowerShell(管理员)(A)”。

  2. 在 Windows PowerShell 中，键入以下 (删除游戏服务) 命令并键入 Enter：get-appxpackage Microsoft.GamingServices | remove-AppxPackage -allusers。

  3. 在同一窗口中，键入以下 (下载游戏服务) 命令并键入 Enter：start ms-windows-store://pdp/?productid=9MWPM2CQNLHN。

  4. 在该页面上安装应用，并在收到用户帐户控制 (UAC) 提示时选择”是“。

  5. 重启设备，然后再次尝试安装该游戏。
